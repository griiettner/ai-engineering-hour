Nunca usar o caractere em dash em nenhum arquivo: HTML, JSON, Markdown ou qualquer outro formato.

**Why:** O usuário considera em dash um marcador óbvio de texto gerado por IA. Quando as pessoas veem esse símbolo, não levam o conteúdo a sério.

**How to apply:** Ao escrever ou editar texto, usar estas alternativas:
- Vírgula (,) ao juntar cláusulas relacionadas
- Dois-pontos (:) ao introduzir uma explicação
- Ponto (.) ao separar sentenças independentes
- Pipe (|) para separadores título/subtítulo
- Middle dot (·) para separadores de lista tipo "Engineering Hour · Part 2"
- Hífen (-) para células de tabela significando "não disponível"
- Parênteses para apartes parentéticos
